# Wavedge

Crypto knowledge platform — know everything about crypto.

Aggregates prices, protocols, news, on-chain data, sentiment, and research into one platform.

## Quick Start

```bash
cp .env.example .env
npm install
npm run dev
```

## Scripts

- `npm run dev` — start dev server with hot reload
- `npm run build` — compile TypeScript
- `npm start` — run compiled output
- `npm test` — run tests

## Health Check

```
GET /health
```
